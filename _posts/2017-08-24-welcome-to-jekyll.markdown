jekyll生成的静态网站，在_post里markdown，上传之后，github服务端会运行jekyll，生成静态网页

_post里markdown命名形式为 `YYYY-MM-DD-name-of-post.ext` ，markdown文件里需要添加jekyll的 `front matter`。可以看看这条日志的源码了解 `front matter`。

Jekyll也提供代码段高亮展示，比如:

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
